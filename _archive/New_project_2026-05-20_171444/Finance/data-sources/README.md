# Data Sources

Monthly statement folders should be copied or referenced here.

Recommended monthly workflow:

1. Create a monthly folder, e.g. `2026-05`.
2. Put all statements for that month in the folder:
   - Fidelity
   - Robinhood
   - Schwab
   - NetBenefits 401(k)
   - Nationwide IUL
   - Chase Checking
   - Amex Savings
   - RSU / ESPP exports
3. Update the dashboard data from the latest folder.
4. Generate a rebalance memo only after the dashboard status is updated.

The current dashboard was created from user-provided statements and exports across these channels.
