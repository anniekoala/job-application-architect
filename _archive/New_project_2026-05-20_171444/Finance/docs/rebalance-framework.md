# Rebalance Framework

## Objective

Increase cash reserve and reduce excessive equity-market exposure without creating unnecessary tax, account, or insurance problems.

## Step 1: Define Cash Target

Choose one:

- 3 months of expenses
- 6 months of expenses
- 12 months of expenses
- Fixed dollar amount

The sell decision should start from the cash target, not from emotions about recent stock prices.

## Step 2: Choose Source Accounts

Prefer:

- Taxable / personal brokerage
- Robinhood individual
- Fidelity individual
- Schwab employer-equity account
- Future RSU / ESPP vesting proceeds

Avoid using as cash-reserve sources:

- 401(k)
- Roth IRA
- HSA
- IUL / insurance policy

Retirement accounts can be rebalanced internally, but they should generally not be withdrawn for cash reserve.

## Step 3: Candidate Sell Order

### 1. Employer Equity Concentration

RBLX should be reviewed first because it combines portfolio risk with employment-income risk.

Possible rule:

- Sell a fixed portion after each vest
- Cap RBLX at a chosen portfolio percentage
- Raise the first cash-reserve tranche from RBLX if tax impact is acceptable

### 2. Duplicate Core Index Exposure

VOO, SPY, and FXAIX overlap heavily.

Decision should depend on:

- Taxable gains
- Account location
- Expense ratio
- Which account should hold the primary core exposure

### 3. Thesis-Weak Smaller Holdings

Review DIS, SPGI, VMACX and any other holding where the current thesis is unclear.

Question:

Would I buy this today with new cash?

If not, it is a cleanup candidate.

### 4. Growth / Tech Size Control

NVDA, QQQ, FBGRX, FDSVX, FNGS, GOOGL, and RBLX all increase growth / tech exposure.

Do not sell only because something has gone up. Instead, decide the intentional allocation band.

## Step 4: Required Data Before Selling

- Official tax lots
- Cost basis
- Holding period
- Unrealized gain / loss
- Account type
- Tax consequences
- Any vesting or lockup context

## Step 5: Output Format

Each rebalance memo should include:

- Cash target
- Amount to raise
- Proposed source account
- Proposed sell candidate
- Reason to sell
- Reason not to sell
- Tax / account concern
- Final action: Sell, Trim, Hold, Wait for data
